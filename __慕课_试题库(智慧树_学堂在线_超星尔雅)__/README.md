## 关于期末考试

超星尔雅是线上考试. 内容是**平时的测试题目**, 建议考试前建立一个平时题目的文档, 考试时直接Ctrl+F查找. (超星考试**答题时间越短, 分越高**)

智慧树的考试是在学校机房考. 机考闭卷, **超出考试时段不可缓考**. 考试**内容不定**, 存在非平时内容的考试.
